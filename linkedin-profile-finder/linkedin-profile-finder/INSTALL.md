# Quick Installation Guide

## 5-Minute Setup

### 1. Load the Extension (2 minutes)

1. Open Chrome and go to: `chrome://extensions/`
2. Turn ON **"Developer mode"** (top-right toggle)
3. Click **"Load unpacked"**
4. Select the `linkedin-profile-finder` folder
5. Done! The extension icon will appear in your toolbar

### 2. Test It Out (3 minutes)

1. Go to LinkedIn: https://www.linkedin.com
2. Search for any job (e.g., "software engineer")
3. Click on a job posting to open it
4. Click the **LinkedIn Profile Finder** extension icon
5. Click **"Find Profiles"**
6. Click the search links to find profiles!
